== Description ==
The LatPay Direct Payment Gateway plugin for Woocommerce allows you to accept the payments directly from your Wordpress website, seamlessly integrating with our secure Level 1 PCI compliant payment platform. Customers who are checking out online will remain on the same page utill the transaction completed and customers those who are all having account in the merchant site they can opt to choose save card for future purchase, increasing conversion rates and buyer confidence. All transactions are screened for fraud in real time to ensure your business stays ahead of the game.

== Installation ==
The easy way:
1. Go to the Plugins Menu in WordPress
2. Search for  Latpay direct 
3. Click  Install 
4. Activate the plugin
5. Go to woocommerce setting->checkout->Latpay
6. Enter the configuration details given by latpay.

Manual Installation:
1.Download the plugin file from wordpress repository.
2.Upload the  woocommerce-lps-hps-payment-gateway  folder to the  /wp-content/plugins/  directory
3.Activate the  woocommerce-lps-hps-payment-gateway  plugin through the  Plugins  menu in WordPress.
